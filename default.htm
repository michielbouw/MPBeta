Webdesign

Bij webdesign kan men aan veel verschillende dingen denken, afhankelijk van wat het doel is van de website kan hierbij een geschikt platform gebruikt worden. Enkele mogenlijkheden worden hieronder verder toegelicht.1


HTML/CSS

Wij bouwen HTML gebaseerde website's die werken met elke mogelijke internet browser. Op ieder apparaat zal de website volledig werken.

Voor onze website's maken wij gebruik van de nieuwste snufjes op het gebied van codering. Wij maken daarom ook gebruik van de nieuwste versie van HTML en CSS. Respectievelijk HTML5 en CSS3.

Responsive design

Onze website's zijn volledig resposive. Dat wil zeggen dat de website's zich aanpassen aan de schermgrootte van het door u gebruikte apparaat. Bij kleine schermen, denk aan een smartphone of tablet, zal de webpagina gedeeltelijk reorganiseren om alles goed leesbaar te houden.

Dus ook mobiel zullen onze website's volledig werken. Wat voor een merk u ook heeft.

WordPress

WordPress is een persoonlijk publicatieplatform gericht op gebruiksgemak. Oorspronkelijk ontworpen voor een weblog, maar uitgegroeid tot een zeer uitgebreid systeem voor websites.

WordPress is een contentmanagementsysteem (CMS) waardoor de gebruiker op een relatief eenvoudige wijze de inhoud (content) van een website kan wijzigen en/of aanvullen zonder dat daarvoor technische kennis vereist is.

Maatwerk

Wij maken voor u een web oplossingen of een webapplicatie. Volledig aangepast aan uw wensen, met een op maat gemaakt design. Maatwerk ook met WordPress CMS mogelijk!
